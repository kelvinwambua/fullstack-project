## How to run this app

<h1> Inititalize the Project</h1>

```
yarn
```

```
npm init
```

<h2> Run the frontend </h2>

```
cd web
yarn dev
```

<h2>Start Up the server</h2>

```
cd server
yarn watch
```
```
yarn dev
```
